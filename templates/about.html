Sobre este Proyecto

Este es un pequeño proyecto de práctica creado para experimentar y adquirir experiencia en la programación y el diseño de una página web con Flask, HTML y CSS. El objetivo es construir una plataforma funcional de venta de autos, explorando el manejo de rutas, plantillas y datos dinámicos.
